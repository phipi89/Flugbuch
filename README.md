# Flugbuch

An experimental, work in progress (!), interactive paragliding flight log viewer.

<video src="https://github.com/user-attachments/assets/a6fa243a-081a-46a1-857c-351fe3f5e71d" autoplay loop muted></video>

## Features

- Real-time flight playback with current altitude, AGL, speed, and vario
- swissALTI3D and swissimage terrain info
- physical shading based on live sun position

## Run

```bash
uv run python -m flugbuch.dashboard
```
